This file contains the final version of scheduling algorithm with nn model and server programing which connects to the mobile.
Here free time of user is stored in free_time.csv nn model uses this free_time.csv to predict the weeks schedule whihc is stored in
predictedValue.csv. topredict.csv contains all the intervals in a week required to find out the schedule. tasks.csv contains the data of 
tasks of user whihc is used to scheduledAlgo.py the scheduled tasks are stored in scheduled_Task.json
